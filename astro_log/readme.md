# AstroLog

*A secure astronomy observation platform.*

---

## Overview

AstroLog is an astronomy observation platform designed for amateur astronomers and astrophotographers to securely record, organize, and revisit their observing sessions.

The current implementation is being developed as a **Minimum Viable Product (MVP)** using **Streamlit** and **Supabase**. This prototype focuses on validating the core functionality, user experience, and system architecture.

The long-term vision is to evolve AstroLog into a production-ready full-stack web application with a dedicated frontend, Python backend, and expanded capabilities.

---

## Features

- Secure User Authentication
- Personal User Profiles
- Observation Logging
- Observation History
- Search & Filtering
- Image Uploads
- Interactive Dashboard
